# Calendar

Simple calendar with a form

## Usage

1. `git clone git@github.com:rupasov/calendar.git`
2. `cd calendar`
3. `npm i`
4. `npm start`
5. `cd server`
6. `npm i`
7. `start MqSQL server`
8. `npm run dev`
